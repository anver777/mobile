# NEON// FINANCE — трекер доходов, целей и заметок

Полностью статический сайт на **чистом HTML + CSS + Vanilla JavaScript** (без
React, без сборщика, без зависимостей). Кибер-неоновый дизайн: неоновая
сетка-плоскость, синтвейв-закат, глитч-заголовки, шестиугольные акценты,
магента + циан + лайм.

## Возможности

- **Доходы** — сумма, источник, категория, дата, заметка. Поиск, фильтр
  периода (все / этот месяц / прошлый), фильтр по категориям (чипы),
  подсчёт суммы отфильтрованного.
- **Цели** — прогресс-бар со свечением, срок, цвет; пополнение и снятие;
  редактирование, счётчик достигнутых целей и общая сумма накоплений.
- **Заметки** — заголовок, текст, цвет, закрепление (pin) сверху, поиск.
- **Аналитика** — 4 карточки (всего, среднее в месяц, лучший месяц, число
  записей), разбивка по категориям (неоновые бары) и столбчатый график по
  месяцам.
- **Настройки** — имя, валюта (₽/$/€/₸/₴), экспорт/импорт JSON, полный сброс.
- **Мобильная версия** — нижняя навигация (dock), модалки-листы снизу
  экрана, поддержка safe-area, крупные тап-цели.

## Как скачать и открыть

Весь сайт лежит в папке **`public/`** — три файла:

```
public/
  index.html   ← открой этот файл двойным кликом
  styles.css
  app.js
```

Достаточно скачать эти три файла — сайт работает офлайн, данные сохраняются
в `localStorage` браузера.

## Как задеплоить

### На GitHub Pages
1. Создай репозиторий и залей содержимое папки `public/` в корень.
2. В настройках репозитория → **Pages** → Source: `main` → `/root`.
3. Готово: сайт доступен по адресу вида `https://<user>.github.io/<repo>/`.

### На Netlify
- Вариант 1 (Drag & Drop): открой https://app.netlify.com/drop и перетащи
  папку `public/`.
- Вариант 2 (из Git): подключи репозиторий, укажи **Publish directory**
  как `public` и оставь **Build command** пустой.

### Локально
Открой `public/index.html` в любом браузере двойным кликом — работает
без сервера.

## Что делает Next.js в проекте

Только одно: подхватывает файлы из `public/` и отдаёт их превью-серверу
платформы. Сам сайт написан на Vanilla JS и никак с Next.js не связан.

## Структура

```
public/
  index.html   — вся разметка (шапка, статистика, вкладки, док, модалка)
  styles.css   — кибер-неоновая тема (сетка, солнце, глитч, cards)
  app.js       — вся логика на чистом JS (localStorage, рендер, формы)

src/app/       — минимальная Next.js обвязка для превью
  layout.tsx
  page.tsx     — читает public/index.html и отдаёт его как страницу
  api/health/  — healthcheck для платформы
```
